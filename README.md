## アプリ概要

「手間を省けてより良い旅行に」をコンセプトにした旅行計画アプリです。<br>

- 地図を見ながら旅行の計画を立てることができます。
- 他のユーザーによって作成された旅行計画を見ることができます。

3 週間のチームプロジェクトで作成しました。<br>
